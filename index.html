Introduction

What is HTML?
HTML (HyperText Markup Language) is the standard language used to create and structure content on the web.

What is CSS?
CSS (Cascading Style Sheets) controls the layout and appearance of a web page, including colors, fonts, and spacing.

HTML Basics

What is a tag?
A tag tells the browser what kind of content an element represents. Example: <p> for a paragraph.

What is an element?
An element is a complete HTML structure including its opening tag, content, and closing tag.

What is an attribute?
An attribute provides extra information about an element. Example: the src attribute specifies the image file in an <img> tag.

CSS Basics

What is a selector?
A selector is what you use in CSS to target elements you want to style. For example, p targets all paragraphs and #intro targets an element with that specific ID.
